# `<colors>`
Sets a UI element's colors.

## Attributes

| Attribute | Type     | Required? |
|-----------|----------|-----------|
| `fg`      | String   | No*       |
| `bg`      | String   | No*       |
| `alt`     | String   | No*       |

\* At least one of the three is required.

### `fg`
The foreground color.

### `bg`
The background color.

### `alt`
The alternate color
