# `<var>`
Declares a local variable.

## Attributes

| Attribute   | Type    | Required? |
|-------------|---------|-----------|
| `ctype`     | String  | Yes       |
| `name`      | String  | Yes       |
| `default`   | String  | No        |

### `ctype`
The C type to use for the variable.

### `name`
The name of the variable.

### `default`
The default value of the variable.
